Note

  • You can select course baskets by using MAJ-SUBJECT-YEAR or CORE-YEAR e.g. use MAJ-PHY-3RD, CORE-1ST, etc. Core year labs must be added manually (due to sections).
  • Mobile users: To save the table as an image, scroll the table completely to the right (such that the time-slots are visible), and click the Save Image button. Otherwise, the image will be cropped.
